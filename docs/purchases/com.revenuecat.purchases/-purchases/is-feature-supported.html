isFeatureSupported

@JvmStatic fun isFeatureSupported(feature: String, context: Context, callback: Callback<Boolean>): Unit

Use this method if you want to check if Subscriptions or other type defined in com.android.billingclient.api.BillingClient.FeatureType is supported. This method is asynchronous since it requires a connected billing client.

Parameters

feature - A feature type to check for support. Must be one of com.android.billingclient.api.BillingClient.FeatureType

context - A context object that will be used to connect to the billing client

callback - Callback that will be notified when the check is complete.
